# ❄️ World Cool Refrigeration

A modern HVAC marketplace web application built with Next.js.

## 🚀 Features

- 🛒 HVAC product marketplace
- 🔧 Technician booking system
- 🧊 Service management
- 📦 Cart & checkout UI
- 📱 Responsive design
- 🌙 Dark mode support
- ⚡ Fast Next.js performance

## 🛠️ Tech Stack

- Next.js (App Router)
- React.js
- CSS (Custom Design System)
